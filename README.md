# GayEngine

An open-source game engine that uses GayScript as its main language.
You can modify the engine yourself if you want.

# How do i code in this engine?
You can write code by creating a file named yourScriptName.gayscript.
Then open it in VS Code and do whatever you want.

# What is GayScript?
